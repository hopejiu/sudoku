## 2. 字体 ✅ 已导入

| 资源 | 格式 | 用途 | 状态 |
|------|------|------|------|
| `assets/fonts/msyh.ttc` | TTC | **微软雅黑常规** — 界面中文主字体 | ✅ |
| `assets/fonts/msyhbd.ttc` | TTC | **微软雅黑粗体** — 标题、数字强调 | ✅ |

### 在 Godot 中配置字体（Theme 编辑器内操作）

1. **导入**：`msyh.ttc` / `msyhbd.ttc` 放入 `assets/fonts/` 后，Godot 会自动创建 `.ttc.import` 资源文件
2. **创建 DynamicFont**：
   - 在 `FileSystem` 面板右键 → `New Resource...` → 选择 `DynamicFont`
   - 保存为 `assets/fonts/msyh_regular.tres`
   - 在 Inspector 中设置 `font_data` → `path` 为 `res://assets/fonts/msyh.ttc`
3. **应用到 Theme**：
   - 打开 `themes/classic.theme`（在 Theme 编辑器）
   - 为 `Label`、`Button`、`Panel` 等 Control 类型设置 `font` → 选刚创建的 `msyh_regular.tres`
   - 同理为粗体效果创建 `msyhbd_regular.tres`（或直接复用常规字体 + `bold` 样式）

> **为什么不在 Project Settings 设置默认字体？**  
> Godot 的 `gui/theme/custom_font` 仅作用于 Fallback 层面。本项目使用 Theme 资源驱动 UI，应在每个 `.theme` 文件内设置字体。

## 5. 项目 Theme 配置

- `project.godot` 已设置 `gui/theme/custom="res://themes/classic.theme"`
- 两个 `.theme` 文件均已内嵌 `FontFile` 对 `msyh.ttc` 的引用
- 运行时通过 `ThemeManager.set_theme()` 切换 `classic.theme` / `purple_light.theme`

### 后续仍需手动操作

| 操作 | 原因 |
|------|------|
| 为图标配置 `modulate` 颜色 | 图标颜色需绑定 ThemeManager 信号，在场景脚本中设置 |
| 在 Theme 编辑器中微调颜色/StyleBox | 当前 Theme 为代码生成的默认值，可进入编辑器手动优化 |

> 核心视觉效果（网格、高亮、冲突）已通过 `SudokuGrid.gd` 的 `_draw()` + `ThemeManager` 完成，**无需额外美术资源即可运行**。
