### 角色系统设计方案（已完成头脑风暴，待实现）

**流程**: 数独菜单 → 开启新游戏 → 角色选(3选1) → 难度选(1~20) → 加载 → 游戏
**角色A** 连锁者：完成行/列/宫时自动揭露1空格 + 特效
**角色B** 明镜者：选中格时键盘自动禁用冲突数字
**角色C** 专注者：连续5正确数字 → 揭露1空格 + Combo特效
**布局**: 底部左半立绘130×184px + 右半数字键盘
**难度上限**: 1~16 → 1~20（挖空数 26~66）
**语音**: 25个触发场景预留，以 OGG + JSON 格式提供
**资源预留**: PNG立绘/头像、JSON台词、OGG语音、Live2D可选
