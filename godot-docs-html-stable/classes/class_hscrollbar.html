HScrollBar

Inherits: ScrollBar < Range < Control < CanvasItem < Node < Object

A horizontal scrollbar that goes from left (min) to right (max).

Description

A horizontal scrollbar, typically used to navigate through content that extends beyond the visible width of a control. It is a Range-based control and goes from left (min) to right (max).

Theme Property Descriptions

int padding_bottom = 0 🔗

Padding between the bottom of the ScrollBar.scroll element and the ScrollBar.grabber.

Note: To apply horizontal padding, modify the left/right content margins of ScrollBar.scroll instead.

int padding_top = 0 🔗

Padding between the top of the ScrollBar.scroll element and the ScrollBar.grabber.

Note: To apply horizontal padding, modify the left/right content margins of ScrollBar.scroll instead.
