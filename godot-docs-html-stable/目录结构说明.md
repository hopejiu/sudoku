## 二、为什么文件这么大（2.0 GB）？

### 根因：Sphinx RTD 主题的导航树重复

每个 HTML 文件都**完整嵌入**了整份文档的侧边栏导航树（Table of Contents），包含全部 500+ 类名 + 300+ 教程链接。

以 `classes/class_aabb.html`（1.34 MB）为例：

| 部分 | 大小 | 占比 |
|------|------|------|
| 侧边栏导航树（全站 TOC） | ~1.24 MB | **92%** |
| 该类的实际文档内容 | ~98 KB | **8%** |
| 纯文本（去标签后） | ~168 KB | 12% |

也就是说，**92% 的内容在每一个 HTML 文件里是完全相同的**（只是链接指向的页面不同）。

### 各目录体积明细

| 目录 | 文件数 | 大小 | 占总量 |
|------|--------|------|--------|
| `classes/` | 1,066 | 1.3 GB | 65% |
| `tutorials/` | 393 | 506 MB | 25% |
| `engine_details/` | 61 | 80 MB | 4% |
| `getting_started/` | 33 | 43 MB | 2% |
| `about/` | 7 | 9 MB | < 1% |
| `community/` | 6 | 7.7 MB | < 1% |
| 其他（index, search 等） | ~5 | ~6 MB | < 1% |

### 压缩效果

| 格式 | 大小 |
|------|------|
| 原始（HTML） | ~2.0 GB |
| tar.gz（classes + tutorials） | **~138 MB**（压缩比约 14:1） |

如此高的压缩比正是因为**导航树的重复度极高**，gzip 可以轻松压缩。

### 建议

- **AI 查询时**：直接使用 HTML 文件即可，AI 可解析。
- **如果空间紧张**：可以对 `classes/` 和 `tutorials/` 做 gzip 压缩存储，查询时解压。
- **已清理的目录**：`_downloads/`、`_images/`、`_sources/`、`_static/` 已被删除，这些是纯资源/重复内容。

## 三、AI 查询可用性分析

### ⛔ 不适合 AI 查询的目录（4 个）

| 目录 | 原因 |
|------|------|
| `_downloads/` | 含二进制下载资源（示例项目压缩包等），AI 无法解析 |
| `_images/` | 纯图片文件（.webp / .png），无文本可供检索 |
| `_sources/` | RST 源文件，内容与 HTML 完全重复，保留一个即可 |
| `_static/` | 样式（CSS）、脚本（JS）、字体（woff2），与文档内容无关 |

### ✅ 适合 AI 查询的目录（6 个）

| 目录 | HTML 数 | 说明 |
|------|---------|------|
| `classes/` | **500+** | ★ **最高频使用**。API 类参考，开发中最常查询 |
| `tutorials/` | **~345+** | ★ **高频使用**。具体功能教程，覆盖面最广 |
| `engine_details/` | **~61** | 引擎原理、编译、调试、性能分析等深度内容 |
| `getting_started/` | **33** | 新手入门，包含 2D/3D 游戏完整范例 |
| `about/` | **7** | FAQ、特性列表、系统要求等基础信息 |
| `community/` | **6** | 社区贡献指南、素材库使用说明 |

### 按使用频率排序

```
classes/        >  tutorials/   >  engine_details/   >  getting_started/   >  about/   >  community/
（API 参考）      （功能教程）      （引擎原理）          （入门指南）            （关于）      （社区）
```

## 四、针对当前项目（Sudoku / Godot 4.x）的优先级建议

当前项目为 **2D 数独游戏**，使用 **GDScript**，以下目录尤为重要：

| 优先级 | 目录 | 原因 |
|--------|------|------|
| ⭐⭐⭐ | `classes/` | 随时查 Node/Control/Resource 等 API |
| ⭐⭐⭐ | `tutorials/ui/` | 2D 界面控件（按钮、容器、主题等） |
| ⭐⭐⭐ | `tutorials/scripting/gdscript/` | GDScript 语法与用法 |
| ⭐⭐ | `tutorials/2d/` | 2D 游戏通用技巧 |
| ⭐⭐ | `tutorials/inputs/` | 输入处理（触摸、键盘等） |
| ⭐⭐ | `tutorials/best_practices/` | 代码组织与性能建议 |
| ⭐ | `getting_started/step_by_step/` | Godot 基础操作 |
| ⭐ | `tutorials/audio/` | 音效播放 |
| ⭐ | `tutorials/editor/` | 编辑器使用技巧 |
